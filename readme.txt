Add support for admin.css @ THEME_DIR (normal or child)

== Description ==
Add support for admin.css @ THEME_DIR (normal or child). Just create admin.css file at "theme_dir"...

== Installation ==
1. Upload `pryc-wp-add-support-for-admincss` dir to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
